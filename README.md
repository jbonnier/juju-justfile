# Juju's Just Recipes

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg?style=flat-square)](LICENSE)

A collection of automation recipes for cloud operations and developer workflows, powered by [just](https://github.com/casey/just).

## Requirements

- [just](https://github.com/casey/just)
- Bash shell
- Cloud CLIs: `az`, `aws`, `jq`, `docker`

## Usage

Display help and available recipes:

```shell
just help
```
